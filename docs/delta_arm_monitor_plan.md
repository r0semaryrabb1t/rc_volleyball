# Delta 机械臂监控 UI 升级 — 最终方案 (plan.md)

**日期**: 2026-03-14
**状态**: 专家委员会全票通过

---

## 1. 目标

将 `delta_arm_monitor_node.cpp` 从 `RCLCPP_INFO_THROTTLE` 单行日志升级为 ANSI 终端全屏动态刷新 UI，风格对标 `motor_monitor_node.cpp`。

## 2. 技术方案

### 2.1 终端控制

- ANSI 转义序列：清屏(`\033[2J`)、光标归位(`\033[H`)、隐藏光标(`\033[?25l`)
- Unicode Box Drawing 字符绘制表格
- 颜色代码：与 motor_monitor_node 保持统一

### 2.2 刷新频率

- **10Hz** (100ms 定时器)，相比 motor_monitor 的 100Hz 更节省资源

### 2.3 布局设计

```
╔═══════════════════════════════════════════════════════════════╗
║                 Delta 机械臂实时监控                          ║
╚═══════════════════════════════════════════════════════════════╝

【机械臂状态】
┌────────┬──────┬──────────┬──────────┬──────────┬──────────┐
│ 状态   │ 模式 │ 当前高度 │ 目标高度 │ 当前角度 │ 目标角度 │
├────────┼──────┼──────────┼──────────┼──────────┼──────────┤
│ ██在线 │ 高度 │  0.200m  │  0.250m  │  45.0°   │  50.0°   │
└────────┴──────┴──────────┴──────────┴──────────┴──────────┘

【电机参数】
┌──────────┬──────────┬──────────┬──────┬──────┐
│ 位置(rad)│ 速度(r/s)│ 力矩(Nm) │ 温度 │ 错误 │
├──────────┼──────────┼──────────┼──────┼──────┤
│   0.785  │   0.12   │   0.50   │ 35°C │   0  │
└──────────┴──────────┴──────────┴──────┴──────┘

指令: mode=高度  目标=0.250m  速度=0.80  更新: 0.1s前

状态频率: 200.0Hz | 指令频率: 50.0Hz | 心跳超时: 500ms | Ctrl+C 退出
```

### 2.4 心跳检测

- 500ms 超时阈值
- `isOnline()` = `msg.online && (now - last_update) < 500ms`
- 上线/离线状态变化时输出 RCLCPP_INFO/WARN 日志

### 2.5 频率统计

- 命令频率 + 状态频率分别统计
- 每秒计算一次

### 2.6 颜色规则

| 条件 | 颜色 |
|------|------|
| 在线 | 绿色 |
| 离线 | 红色 |
| 错误码 != 0 | 红色 |
| 温度 > 50°C | 黄色 |
| 模式标签 | 青色 |
| 等待数据 | 灰色(DIM) |

### 2.7 模式映射

| 值 | 中文 |
|----|------|
| 0  | 空闲 |
| 1  | 高度 |
| 2  | 回零 |
