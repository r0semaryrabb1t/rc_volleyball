# GO-M8010-6 力位混合控制指南

## 1. 控制原理

电机内部使用 PD 控制器，将位置偏差和速度偏差反馈叠加到力矩输出上：

```
τ = τ_ff + K_p × (p_des - p) + K_d × (ω_des - ω)
```

| 符号 | 含义 | SDK 字段 | 单位 |
|------|------|---------|------|
| τ_ff | 前馈力矩 | `cmd.T` | N.m（转子侧） |
| p_des | 期望角度位置 | `cmd.Pos` | rad（转子侧，多圈累加） |
| ω_des | 期望角速度 | `cmd.W` | rad/s（转子侧） |
| K_p | 位置刚度 | `cmd.K_P` | — |
| K_d | 速度刚度（阻尼） | `cmd.K_W` | — |
| p | 当前角度位置（反馈） | — | rad |
| ω | 当前角速度（反馈） | — | rad/s |

> **重要：** 所有指令参数均为**电机转子侧**，减速比为 **6.33**。
> 输出轴目标值需乘以 6.33 转换为转子侧值。
> 例：输出端目标 3.14 rad → `cmd.Pos = 3.14 * 6.33 = 19.88`

## 2. 参数范围

| 参数 | 范围 | 说明 |
|------|------|------|
| τ_ff | \|τ_ff\| ≤ 127.99 N.m | 前馈力矩 |
| ω_des | \|ω_des\| ≤ 804.0 rad/s | 期望角速度 |
| θ_des | \|θ_des\| ≤ 411774 rad（约65535圈） | 期望位置（多圈累加） |
| K_p | 0 ≤ K_p ≤ 25.599 | 位置刚度 |
| K_d | 0 ≤ K_d ≤ 25.599 | 速度刚度/阻尼 |

## 3. 各控制模式参数设置

力位混合控制的各子模式，本质上是公式中不同项被置零的特例。

### 3.1 纯位置模式

电机输出轴稳定在指定位置。

```cpp
cmd.T   = 0.0;
cmd.W   = 0.0;
cmd.Pos = 目标位置(rad) * 6.33;   // 转子侧
cmd.K_P = 0.02;                   // 推荐起步值
cmd.K_W = 0.0;
```

**推荐值：**
| 参数 | 推荐值 | 说明 |
|------|--------|------|
| K_P | 0.02 ~ 0.1 | 从小值开始调，避免瞬间大力矩 |
| K_W | 0.0 ~ 0.02 | 可选，加入阻尼减少振荡 |

> 注意：若 Pos 与当前位置差距很大，电机产生的力矩会很大，可能触发电源保护。
> 建议让 `cmd.Pos` 缓慢变化，避免产生瞬间极大力矩。

### 3.2 速度模式

电机输出轴以固定速度旋转。

```cpp
cmd.T   = 0.0;
cmd.W   = 6.28 * 6.33;   // 输出轴 1 转/秒 → 转子侧
cmd.Pos = 0.0;
cmd.K_P = 0.0;            // 必须为 0
cmd.K_W = 0.02;           // 推荐起步值
```

**推荐值：**
| 参数 | 推荐值 | 说明 |
|------|--------|------|
| K_W | 0.02 ~ 0.1 | 速度比例系数，从小值开始 |
| K_P | **必须为 0** | 否则不是纯速度模式 |

### 3.3 阻尼模式

速度模式的特殊情况，W=0 时电机保持转轴速度为 0，被外力旋转时产生反向阻抗力矩。

```cpp
cmd.T   = 0.0;
cmd.W   = 0.0;
cmd.Pos = 0.0;
cmd.K_P = 0.0;
cmd.K_W = 0.02;   // 阻尼系数
```

**推荐值：**
| 参数 | 推荐值 | 说明 |
|------|--------|------|
| K_W | 0.02 ~ 0.5 | 越大阻尼越强 |

### 3.4 力矩模式

电机持续输出恒定力矩。空转时会持续加速到最大转速。

```cpp
cmd.T   = 0.05;   // 无负载安全起步值
cmd.W   = 0.0;
cmd.Pos = 0.0;
cmd.K_P = 0.0;
cmd.K_W = 0.0;
```

**推荐值：**
| 参数 | 推荐值 | 说明 |
|------|--------|------|
| T | 0.05 ~ 0.2 | 无负载先用小值测试 |

> 注意：空转时给大力矩，电机会持续加速，请小心使用。

### 3.5 零力矩模式

力矩为 0，电机仅靠自身摩擦力矩抵抗，手动转动阻力明显小于断电状态。

```cpp
cmd.T   = 0.0;
cmd.W   = 0.0;
cmd.Pos = 0.0;
cmd.K_P = 0.0;
cmd.K_W = 0.0;
```

### 3.6 力位混合模式（完整）

四足机器人等实际控制中最常用的模式，**同时**发送前馈力矩、目标角度和目标角速度。

```cpp
cmd.motorType = MotorType::Go2;
cmd.id   = 0;
cmd.mode = 1;              // FOC 闭环模式（必须为 1）
cmd.T    = 0.0;            // 前馈力矩 τ_ff
cmd.Pos  = 3.14 * 6.33;   // 期望输出轴位置 3.14 rad → 转子侧
cmd.W    = 0.0;            // 期望角速度
cmd.K_P  = 0.02;           // 位置刚度
cmd.K_W  = 0.02;           // 速度刚度/阻尼
_ioPort.sendRecv(&cmd, &data);
```

**推荐起步值：**
| 参数 | 推荐值 | 调参方向 |
|------|--------|----------|
| T | 0.0 ~ 0.1 | 根据负载重力补偿需求设置前馈 |
| K_P | 0.02 ~ 0.1 | 从小值开始，逐步增大直到位置跟踪满意 |
| K_W | 0.02 ~ 0.1 | 从小值开始，增大可减少振荡、增加阻尼 |
| Pos | 当前位置附近 | 避免突变，建议做轨迹插值 |
| W | 0.0 | 有速度规划时可设为目标速度 |

**调参建议：**
1. 先将所有参数置零（零力矩模式），确认电机通信正常
2. 小 K_P 开始，观察位置跟踪效果
3. 若出现振荡，增大 K_W（阻尼）
4. 若有重力负载，加入前馈力矩 T 做重力补偿
5. K_P 和 K_W 都从 0.02 起步，每次增量不超过 0.02

## 4. mode 字段说明

| 值 | 含义 |
|------|------|
| 0 | 锁定（Default），5个控制参数无效 |
| 1 | FOC 闭环，**力位混合控制必须设为 1** |
| 2 | 编码器校准（等待 5s，期间不可发送数据包） |

## 5. 反馈数据

电机返回 16 字节数据包，关键反馈量（均为转子侧）：

| 字段 | 含义 | 换算 |
|------|------|------|
| τ_fbk | 实际输出转矩 | `τ(N.m) = τ_fbk / 256` |
| ω_fbk | 实际输出速度 | `ω(rad/s) = ω_fbk / 256 × 2π` |
| θ_fbk | 实际输出位置（多圈累加） | `θ(rad) = θ_fbk / 32768 × 2π` |
| TEMP | 电机温度 | 摄氏度，90°C 触发温度保护 |
| MERROR | 错误标识 | 0正常 1过热 2过流 3过压 4编码器故障 |

## 6. 通信要点

- 接口：RS-485，波特率 **4.0 Mbps**，8bit 数据位，无校验，1 停止位
- 总线最多 15 个电机（ID 0~14），ID 不可重复
- 供电电压：DC 24V（最大允许 DC 30V）
- 减速比：**6.33**
